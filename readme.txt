[url=https://steamcommunity.com/sharedfiles/filedetails/?id=1672477224]ZMM More Variants[/url]と[url=https://steamcommunity.com/sharedfiles/filedetails/?id=1138099675]M_Variants_JP+[/url]を組み合わせて少しバランスを調整したModです。

[h1]概要[/h1]
- 装備の改造の上限を99にします。
- 一部の改造による性能変化を変更します。
- 全ての改造可能な装備に「生産コスト軽減」の改造を追加します。
- 全ての改造可能な装備の旧式装備からの更新が可能になります。
- 以下の装備を改造可能にします。
  - 歩兵装備
  - 支援装備
  - 野砲
  - 対戦車砲
  - 対空砲
  - 装甲車
  - 兵員輸送車
  - 自動車
  - 輸送機
  - ロケット砲
  - 自動車化ロケット砲(カチューシャ)

[h1]類似Modとの関係性[/h1]
日本語化版のMore Variantsが更新が止まっていたので、最新版に対応していたZMM More Variantsをベースに日本語化し、日本語化版に寄せてバランス調整を加え、日本語化版の独自機能であった全装備の更新対応も行いました。
なお、海軍はMtGにより改造そのものが無くなっているため非対応です。また、輸送船は改造したら在庫表示がなんかおかしかったのでバグる可能性を考慮して除外してます。
